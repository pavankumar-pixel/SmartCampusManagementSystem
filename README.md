# Smart Campus Management System

A Java + MySQL console-based project.

## Technologies Used
- Core Java
- JDBC
- MySQL

## Modules
- Student Management
- Faculty Management
- Attendance Management
- Complaint Management
- Event Management
- Login System

## How to Run
1. Import the SQL file into MySQL Workbench
2. Download mysql-connector-j JAR and place in project folder
3. Compile: javac -cp ".;mysql-connector-j-9.3.0.jar" *.java
4. Run: java -cp ".;mysql-connector-j-9.3.0.jar" MainMenu
5. Login with username: admin, password: admin123
